# AI Resume Matcher

## Description
AI-based Resume and Job Description matching using ASP.NET Core MVC and Ollama.

## Features
- Upload Resume
- Paste Job Description
- AI Matching (Free LLM via Ollama)

## Setup

### Run App
dotnet run

### Setup Ollama
- Install Ollama
- Run: ollama pull mistral
- Start: ollama serve

## AWS Deployment
- Use EC2 Free Tier
- Open port 11434
